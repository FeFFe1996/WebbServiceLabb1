To use this application, you need to first
get an openrouter api key from https://openrouter.ai/.

Create an account if you don't have one and create a new API key.
Place the key in an enviroment variable called OPENROUTER_API_KEY.

This webservice uses a free model for the calls but if you want to use another one, 
you can update this with a model of you choice in the service class on line 31.
